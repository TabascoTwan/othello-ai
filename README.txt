1. Start by extracting the zip file to preferred location
2. Open the project in preferred IDEA (we recommend Intellij)
3. When in the project go to the following location Othello-FX-master/Othello FX Edu/src/main and open the class AgentManager.java
4. On line 46 you will find the class called AgentManager, next to the line number you will be able to press a green triangle and then press run 'AgentManager' and the game will start.
5. Enjoy!
